# Produce-Mode Discovery on a Payments Platform

## Problem/Feature Description

Run produce-mode discovery on a payments-platform company that ships:
- Multi-language SDKs with releases in the last 30 days
- A public OpenAPI spec
- An officially-hosted MCP server endpoint
- A public agent-toolkit repo (~1.5k stars) integrating with major agent frameworks
- An official "agent skills" repo with at least two published skills (best-practices guide, version-upgrade helper)

Open access tier, signed-webhook event surface with known integrator failure modes.

Produce a discovery.json conforming to the produce-mode v3 contract.
