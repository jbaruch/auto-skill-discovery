# Target Selection for DataBridge

## Problem/Feature Description

A discovery pipeline run has completed for DataBridge, a data infrastructure company. The discovery output is available at `inputs/discovery.json`. A product manager wants to understand which engineering targets are available and move forward with the selection process.

Process this discovery output using the target selection workflow and produce a written summary of what was determined. Write your output to a file called `selection-output.md`.

## Output Specification

- `selection-output.md` — a markdown file containing the complete output of the target selection workflow, including any messages, lists, or guidance produced during the process.
